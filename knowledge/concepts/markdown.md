# Markdown

## What is it?

Markdown is a lightweight markup language used to format plain text. It is commonly used for documentation, README files and technical notes.

---

## Why is it Important?

Markdown allows documentation to be written quickly while remaining readable as plain text. Platforms such as GitHub automatically render Markdown into formatted documents.

---

## How do I use it?

Some examples include:

```markdown
# Heading

## Subheading

- Bullet point

- [ ] Checklist

`inline code`

```python
print("Hello")
```
```

Markdown files use the `.md` file extension.

---

## Key Takeaways

- Markdown is plain text
- It is easy to write and read
- GitHub automatically renders Markdown
- It is widely used for technical documentation
