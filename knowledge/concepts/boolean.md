# Boolean

## What is it?

A Boolean is a data type that represents one of two possible values:

- `True`
- `False`

Boolean values are commonly used to represent the result of a question or comparison.

---

## Why is it Important?

Programs frequently need to make decisions, a Boolean allows Python to represent whether a condition is true or false.

---

## How do I use it?

A Boolean can be stored in a variable

```python
is_adult = True
```

More commonly, Booleans are produced by comparison operators

```python
is_adult = age >= 18
```

The result is either:

```python
True
```

or

```python
False
```

---

## Key Takeaways

- A Boolean has only two possible values
- `True` and `False` are values, not strings
- Comparisons return Boolean values
- Booleans are commonly used when making decisions
