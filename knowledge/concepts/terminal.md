# Terminal

## What is it?

The terminal is a text-based interface used to communicate with the operating system of a computer by entering commands. It allows programs such as Python and Git to be started from the command line and displays any output they produce.

---

## Why is it Important?

Many development tools are designed to be used from the terminal. Learning how to use the terminal provides greater control over the system itself and is an essential skill for software engineering, DevOps and cloud engineering.

---

## How do I use it?

A termal can be used to start programs with commands

```powershell
git status
```

The terminal starts the requested program and displays the output

---

## Key Takeaways

- A terminal is an application
- It provides access to a command-line environment
- It can start programs and display their output
- It does not execute Python code itself
