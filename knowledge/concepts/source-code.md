# Source Code

## What is it?

Source code is the human-readable text written by a programmer using a programming language such as Python. It contains the instructions that describe what a program should do.

---

## Why is it Important?

Source code is the starting point for every software application. Before a program can run, the source code must be processed by a language-specific tool such as the Python interpreter.

---

## How do I use it?

Source code is written using a text editor or an Integrated Development Environment (IDE) such as Visual Studio Code. Once written, it can be executed using a language interpreter.

Example:

```python
print("Hello, World!")
```

---

## Key Takeaways

- Source code is written by developers
- It is stored in text files
- The interpreter executes the source code
- Source code can be modified and run repeatedly
