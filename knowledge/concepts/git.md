# Git

## What is it?

Git is a distributed version control system used to track changes made to files over time. Rather than creating multiple versions of the same file, Git records changes as commits.

---

## Why is it Important?

Git provides a complete history of a project, allowing developers to experiment safely, restore previous versions and collaborate without overwriting each other's work.

---

## How do I use it?

Some common commands include:

```powershell
git clone <repository-url>
git status
git add .
git commit -m "Commit message"
git push
```

These commands allow me to clone repositories, stage changes, create commits and upload my work to GitHub.

---

## Key Takeaways

- Git tracks files and their history
- A commit represents a snapshot of a project
- Local repositories can be synchronised with remote repositories such as GitHub
