# Python Interpreter

## What is it?

The Python interpreter is the program that executes Python code. When a Python file is run, the interpreter reads the source code, understands each instruction and performs the requested actions.

---

## Why is it Important?

The interpreter is responsible for turning Python source code into a running program, without an interpreter no Python programs can be read or executed as the computer will just see a text file.

---

## How do I use it?

The interpreter is usually started automatically when called by VS Code or the terminal

For example:
```powershell
python hello.py
```

The interpreter will:

1. Open the file
2. Read each instruction 
3. Execute instructions in order
4. Display output or errors to the terminal

---

## Key Takeaways

- The interpreter is a separate program from VS Code
- Python programs cannot run without a Python interpreter
- The interpreter executes Python code one instruction at a time from top to bottom
- Output/errors returned to terminal
