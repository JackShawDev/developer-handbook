# Command Line Interface (CLI)

## What is it?

A Command Line Interface (CLI) is a way of interacting with a computer by typing commands instead of using a graphical interface. Commands are entered into a terminal and executed by a shell.

---

## Why is it Important?

Many development tools are designed to be used from the command line. Understanding the CLI makes it easier to use Git, Python and many other software development tools.

---

## How do I use it?

Examples of commands used so far include:

```powershell
pwd
cd
dir
code .
python hello.py
git status
```

Each command performs a specific task such as changing directories, running Python or checking Git status.

---

## Key Takeaways

- The CLI is a method of interacting with a computer
- Commands are entered into a terminal
- Different commands perform different tasks
- The CLI is widely used in software engineering
